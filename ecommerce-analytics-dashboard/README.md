# E-Commerce Analytics Dashboard

A recruiter-ready e-commerce analytics dashboard project built for a Data Analyst / BI portfolio.

## Live Dashboard

After uploading this folder to your GitHub Pages portfolio repository, the dashboard will be available at:

```text
https://codewith-atif.github.io/portfolio/ecommerce-analytics-dashboard/
```

## Project Overview

This project analyzes online sales performance using a realistic e-commerce dataset. It focuses on revenue, profit, orders, customer segments, categories, products, regional performance, payment behavior and returns.

## Key Features

- Total Revenue KPI
- Total Profit KPI
- Total Orders KPI
- Average Order Value KPI
- Return Rate KPI
- Profit Margin KPI
- Monthly Revenue Trend
- Revenue by Category
- Regional Sales Performance
- Customer Segment Analysis
- Top Products by Revenue
- Payment Method Distribution
- Interactive filters

## Tools Used

HTML, CSS, JavaScript, Plotly.js, SQL, CSV dataset, GitHub Pages
