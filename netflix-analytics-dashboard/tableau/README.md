# Tableau build guide

Connect Tableau Public to `data/netflix_titles.csv`. Create sheets for: KPI tiles, yearly catalog trend, content-type donut, top genres, rating distribution, and country ranking. Add `Type`, `Country`, `Rating`, and `Release Year` as dashboard filters and apply them to all worksheets. Use Netflix red `#E50914`, charcoal `#141414`, and off-white `#F5F5F1`. Publish to Tableau Public and place the public URL in the main project README.
