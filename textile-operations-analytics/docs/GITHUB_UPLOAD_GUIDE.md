# Upload to GitHub Without VS Code

## Upload the project

1. Download and unzip `textile-operations-analytics.zip` on your computer.
2. Open `https://github.com/Codewith-Atif/portfolio` and sign in.
3. Click **Add file → Upload files**.
4. Open the unzipped folder. Drag the complete `textile-operations-analytics` folder into the upload area. If your browser does not accept the folder, create that folder in GitHub first and upload its contents in smaller groups.
5. In Commit changes, write: `Add textile operations analytics project`.
6. Choose **Commit directly to the main branch** and click **Commit changes**.

## Add the project card to your portfolio

Edit the main portfolio `index.html`. Inside `<div class="project-grid">`, add:

```html
<div class="project-card">
  <h3>Textile Operations Analytics</h3>
  <div class="project-tags">
    <span>Python</span><span>SQL</span><span>Excel</span>
    <span>Tableau</span><span>JavaScript</span>
  </div>
  <p>End-to-end textile manufacturing analysis covering production efficiency, quality, waste, downtime and delivery KPIs.</p>
  <div class="project-actions">
    <a href="./textile-operations-analytics/index.html" target="_blank" class="btn primary">Live Demo</a>
    <a href="https://github.com/Codewith-Atif/portfolio/tree/main/textile-operations-analytics" target="_blank" class="btn">Code</a>
  </div>
</div>
```

Click **Commit changes**. GitHub Pages normally updates within a few minutes.

Live URL: `https://codewith-atif.github.io/portfolio/textile-operations-analytics/`
