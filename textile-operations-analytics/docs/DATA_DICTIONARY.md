# Data Dictionary

| Field | Meaning |
|---|---|
| run_id | Unique production-run identifier |
| production_date | Date of production |
| plant | Manufacturing location |
| production_line | Line code within a plant |
| shift | Morning, Evening or Night |
| fabric_type | Cotton, Denim, Polyester, Viscose or Blend |
| target_meters | Planned output in metres |
| actual_meters | Recorded output in metres |
| good_meters | First-pass accepted output |
| defect_meters | Output requiring rework or rejection |
| defect_type | Primary defect category |
| material_issued_kg | Raw material issued for the run |
| waste_kg | Recorded material waste |
| downtime_minutes | Unplanned machine downtime |
| machine_speed_rpm | Average machine speed |
| operators_count | Operators assigned to the run |
| energy_kwh | Electricity consumed |
| order_value_inr | Estimated run value in INR |
| delivery_status | On Time or Late |
| quality_grade | A, B or C quality classification |
