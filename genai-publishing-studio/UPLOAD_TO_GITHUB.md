# Upload and connect this project

## Recommended: GitHub website (easiest)

1. Extract `genai-publishing-studio.zip`.
2. Open `github.com/Codewith-Atif/portfolio` and sign in.
3. Click **Add file → Upload files**.
4. Drag the complete `genai-publishing-studio` folder into the uploader. Confirm that GitHub shows paths such as `genai-publishing-studio/index.html`.
5. Commit message: `Add GenAI Publishing Studio showcase`.
6. Click **Commit changes**.
7. Wait 1–3 minutes, then open `https://codewith-atif.github.io/portfolio/genai-publishing-studio/`.

## Command line method

```bash
git clone https://github.com/Codewith-Atif/portfolio.git
cd portfolio
# Copy the extracted genai-publishing-studio folder here
git add genai-publishing-studio
git commit -m "Add GenAI Publishing Studio showcase"
git push origin main
```

## Add the project card to your main portfolio

In the projects section of the main `index.html`, copy an existing project card and use:

- Title: `GenAI Publishing Studio`
- Description: `Multimodal AI production showcase for educational publishing—images, video, voice, animation, lecture workflows and creative analytics.`
- Live link: `genai-publishing-studio/`
- GitHub link: `https://github.com/Codewith-Atif/portfolio/tree/main/genai-publishing-studio`
- Skills: `Prompt Engineering · Gen AI · Python · SQL · Excel · Tableau · JavaScript`

## GitHub Pages check

Your portfolio is already served from the repository. If the new URL does not open, go to **Settings → Pages**, select **Deploy from a branch**, choose `main` and `/ (root)`, then save.
