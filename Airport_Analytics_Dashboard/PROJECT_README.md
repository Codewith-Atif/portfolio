# Airport Analytics Dashboard ✈️

A complete data analytics portfolio project built for airport operational and passenger-flow analysis.

## Project Overview
Built interactive drill-down dashboards using:
- Python (Pandas, Plotly, Streamlit)
- Tableau Dashboard
- Power BI Dashboard

The project focuses on:
- Passenger traffic analysis
- Flight operations monitoring
- Airline performance
- Airport efficiency metrics
- Delay analysis
- Revenue and route insights

## Dashboard Features

### KPI Analysis
- Total passengers
- Total flights
- Average delay
- On-time performance
- Top airlines

### Passenger Analytics
- Monthly passenger trends
- Domestic vs international traffic
- Peak travel periods
- Passenger demographics

### Flight Operations
- Delay patterns
- Route performance
- Airline comparison
- Airport congestion analysis
